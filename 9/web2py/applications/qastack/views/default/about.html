Hanging paragraph CSS style, for APA-style references

CSS for creating a hanging indent —APA style— References list

The following is a quick template that will allow anyone to copy and apply the CSS needed to hang references in their web publications using correct APA style syntax. You may format your references in whatever semantic structure you are using: as list items within an ordered or unordered list, as definition descriptions within a defintion list, or even as paragraphs within a division element.

To apply:
1. Paste the following code into your CSS file: the class="ref-apa" css class to the wrapping element in your references, whether it be a div, an ol, ul or dl.
update: As an alternative to .ref-apa, the following class now work interchangeably too: .refapa, .apa, .aparef, and .apa-ref
