zdotfiles

### 1-First Diminsion :
    Command line
    zsh
    
### 2- Second Diminsion :
     Text Documents
     Vim
     
### 3- Third Diminsion :
     Directory
     File/Folder Manager
     
### 4- Fourth Diminsion :
     Access : Logs
     
### 5- Fivth Diminsion :
     Access : Current processes
### 6- Sixth Diminsion :
     Access : User applications
     Ex: Autokey
     Shortcuts
     
### 7- Seventh Diminsion :
     Access : Root - System Administration
### 8- Eighth Diminsion
     Access : Internet Access your account on-line
     Bookmark
     
### 9- Nineth Diminsion
     Access : SSH/Root
### 10- Multisystem
     Access your System from different OS
