zdotfiles

### 1-First Diminsion :
    Command line
### 2- Second Diminsion :
     Text Documents
### 3- Third Diminsion :
     GUI
### 4- Fourth Diminsion :
     Access : Logs
### 5- Fivth Diminsion :
     Access : Current processes
### 6- Sixth Diminsion :
     Access : User applications
     Ex: Autokey
### 7- Seventh Diminsion :
     Access : Root - System Administration
### 8- Eighth Diminsion
     Access : Internet Access your account on-line
### 9- Nineth Diminsion
     Access : SSH/Root
### 10- Multisystem
     Access your System from different OS

### Manual (not recommended)

1. Install the
   [zdotfiles](https://github.com/tazjel/zdotfiles.git)/
2. Put file in your Home directory (usually `~/zdotfiles/`)
